Como fazemos

Briefing

Para começar é importante que a gente colete todas as informaçoes importantes para o seu site, é o momento de você falar sobre o motivo que o levou a querer um novo site, quais cores gosta, sobre sua segmentação, serviços, produtos, referências entre outras questões.

Mapa Mental

Após o Briefing é a hora de descrevermos todo o conteúdo do site, como serão as páginas e o direcionamento de cada setor. Assim temos uma ideia geral de como será o site e como vamos criar os layouts de acordo com suas necessidades.

Criação de Layout

Nosso designer cria um layour exclusivo para você, o layout já é a cara que seu site vai ter quando estiver online, e nele você vai poder solicitar as alterações para que fique do jeito que você gostar.

Upando o Site

Nosso programador a partir do layout criado e aprovado, vai iniciar a programação do site. Nesse momento é importante que todas as informações já estejam com a gente para que possamos adicionar de forma correta.
